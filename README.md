# Orionjs typescript-starter

## Requirements

Node >= 14 (LTS)

## Environment Variables

copy `.env.example` file to `.env` with command:

```
cp .env.example .env
```

## Install

```
yarn install
```

## Run develop

```cli
yarn dev
```

Open your browser and navigate to [http://localhost:3000/](http://localhost:3000/)

## Build

```cli
yarn build
```
